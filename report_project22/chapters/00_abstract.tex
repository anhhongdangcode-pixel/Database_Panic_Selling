\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This report presents the design and implementation of a data-driven system for detecting and quantifying panic selling behavior among individual investors in the Vietnamese stock market. The system integrates a relational database built on MySQL with a Python-based behavioral analytics pipeline to identify abnormal selling patterns under market stress conditions.

The project generates a synthetic dataset simulating three investor archetypes — FOMO, Rational, and Noise investors — trading across nine Vietnamese stock tickers over the year 2025. Behavioral features including drawdown level, sell spike intensity, and loss sensitivity are extracted from transaction histories and aggregated into a Panic Score for each investor. Advanced database objects including indexes, views, stored procedures, user-defined functions, and triggers are implemented to support efficient querying and automated warning generation.

A lightweight web application built with Flask provides an interactive interface for querying investor profiles, viewing panic scores, and monitoring behavioral warnings in real time.

\vspace{1cm}
\noindent\textbf{Keywords:} panic selling, behavioral finance, synthetic data generation, MySQL, Python, investor profiling, market stress detection
